## 5. 一票否决项

出现以下任一项，本轮不能判定为 phase-1 验收通过：
- [ ] 登录成功但刷新丢会话
- [ ] 前端受保护请求未携带真实用户 JWT
- [ ] 管理写操作前端提示成功但后端未落库
- [ ] `upload-url -> 上传 -> image-url` 任一环节失效
- [ ] follow / profile / plant / journal 主流程出现契约错位
- [ ] 白屏或可复现致命错误
