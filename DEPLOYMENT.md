## 为什么归档

原部署文档默认：
- 根目录直接构建前端
- 根目录脚本直接承担主要部署职责
- 前后端仍按单体方式描述

这与当前三端分离后的实际结构已不完全一致，继续直接照抄可能造成：
- 部署目标目录错误
- API 地址配置错误
- 将根目录误认为当前生产发布入口
