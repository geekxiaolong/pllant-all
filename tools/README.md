# 小工具

## 白底去背 · 批处理 (remove-bg.html)

把**纯白或接近白色的背景**变成透明，得到透明 PNG，**支持多张图片批处理**。适合 fal.ai 生成的植物卡通图等白底插画。

**用法：**

1. 用浏览器直接打开 `remove-bg.html`（双击或 `open tools/remove-bg.html`）。
2. 点击选择多张图片，或一次拖拽多张图片到页面里。
3. 调节「白色阈值」会统一应用到所有图片（默认 250 一般即可）。
4. 每张图可单独点「下载此张」，或点「下载全部透明 PNG」依次下载所有结果。

无需安装，所有处理在本地完成，图片不会上传到任何服务器。
